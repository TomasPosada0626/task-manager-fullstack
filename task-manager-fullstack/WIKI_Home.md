---

**Cómo usar:**
1. Copia el título y la descripción de cada fila.
2. Ve a la pestaña Issues en GitHub y haz clic en "New issue".
3. Pega el título y la descripción, asigna etiquetas si lo deseas.
4. Repite para cada issue relevante para tu proyecto.

Puedes agregar, quitar o modificar issues según las necesidades de tu proyecto.
